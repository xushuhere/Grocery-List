# Extra Requirements

**Author**: Team 16: Anshul Goel, Dwight Nelson, Shu Xu, Zakariya Ahmad

## 1 List of Extra Requirements
* Intuitive User Interface for usability
* Portability to iOS and Windows 10 should be considered during design decisions 
* Data robustness
* Must be programmed with maintainability in mind
* Updates for new Android releases must be timely
* User Inteface must be up to date with Android design requirements
